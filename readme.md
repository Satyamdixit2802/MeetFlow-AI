# MeetFlow AI — AI Meeting Summariser & Action Tracker

MeetMind AI is a full-stack AI-powered meeting assistant that converts meeting recordings into structured, actionable insights.

The application transcribes meeting audio using Groq Whisper Large V3 Turbo, generates concise AI summaries, extracts action items with deadlines, and tracks progress through an interactive dashboard.

<br>


## Features
<li>Audio upload for meeting recordings
<li>Speech-to-text transcription using Groq Whisper
<li>AI-generated meeting summaries
<li>Automatic action item extraction
<li>Action tracker dashboard
<li>Meeting analytics and insights
<li>PDF export of meeting reports
<li>Text-to-speech summary playback
<li>Authentication and user profiles
